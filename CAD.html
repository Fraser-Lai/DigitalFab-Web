Keychain

How to 3D print your own personalize keychain?.

Have your own personalize keychain design in Fusion 360

Go to the file icon and select 3D print

In this case I am using the school 3D printer (cura 2+). Therefore I have to download the app inorder to save the file.

When the file is opened in cura 2+, you need to customize the setting; such as select the printer to cura 2+, matrial used is PLA etc.

After you have done customizing the printer, select slice. It will estimate the time to print. Save your file in a SD card and insert it in the printer.

At the beginning, you have to ensure that the 3D printer prints normally. If the 3D printer function normally, you can leave to do other work and come back when it is done.
